# Project Instructions

Build a routine-first personal dashboard.

Main priorities:
1. Mobile-first responsive design
2. Clean modern dark UI
3. Simple navigation
4. Reusable React components
5. Clear readable code
6. localStorage persistence for version 1

Pages:
- Dashboard
- Routine
- Planner
- Budget
- Reports
- Settings

Rules:
- Keep components small
- Avoid giant files
- Prefer simple solutions
- Do not break existing pages when adding features
- Test after each change
- Use Tailwind for styling
- Use lucide-react for icons
- Use recharts for charts
